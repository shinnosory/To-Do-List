#  Todo List

Este projeto é uma **Lista de Tarefas (Todo List)** desenvolvida com **HTML, CSS e JavaScript**, com foco em praticar lógica básica de programação.

---

##  Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Contador de tarefas
- Interface limpa e organizada
- Layout responsivo

---

## Estilo Visual

- Paleta de cores **pastel (lilás/rosa)**
- Layout centralizado
- Bordas arredondadas
- Efeitos suaves de hover
- Visual leve e moderno

---

## Tecnologias Utilizadas

- **HTML5** — estrutura da aplicação  
- **CSS3** — estilização e layout  
- **JavaScript (Vanilla)** — lógica da lista de tarefas
